# ScraperBeatPort
ScraperBeatPort es una herramienta web sencilla que permite extraer y descargar los listados "Top 100" de Beatport en formato CSV. Diseñada para DJs y coleccionistas, facilita la importación rápida de metadatos de charts (Techno, House, DnB, etc.) a bases de datos personales como Muzbase sin necesidad de copiar manualmente.
